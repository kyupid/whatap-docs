~~초안~~

NMS(Network Management System)는 네트워크 장비(라우터, 스위치, 서버, 방화벽, 무선 액세스 포인트 등)의 상태를 중앙에서 통합적으로 모니터링하여 성능 및 안정성을 유지하는 시스템입니다. 이를 통해 네트워크 트래픽, 장비 상태, 대역폭 사용량 등을 실시간으로 확인하고, 장애 발생 시 빠르게 원인을 파악할 수 있도록 도와줍니다.

NMS는 장애 예방과 문제 탐지에 중점을 두며, 네트워크의 성능 저하를 최소화하고 안정적인 서비스 품질(QoS)을 보장하는 데 필수적인 역할을 합니다. 이로써 조직의 IT 인프라를 효율적으로 운영하고, 비즈니스 연속성을 유지할 수 있도록 지원합니다.

### 주요 기능

* **디바이스 상태 모니터링**
    
    인터페이스의 inbound,  outbound Throughput 측정
    
    CPU, Memory, Session등 OID 에 따라 다양한 지표를 모니터링
    
* **디바이스 대시보드 제공**
    
    인터페이스 상태 및 link up/down 상태 감시
    
* **이벤트 실시간 알림 및 검색**
    
    Trap 및 syslog 메세지를 통한 이벤트 실시간 출력 알림 제공
    
    이벤트 로그 검색 
    

### 주요 특장점

* 국내외 약 20개 이상의 vendor사 모니터링

* MIB 브라우저 기능 제공 및 Upload 기능 제공

* 모니터링 대상 OID 리스트 등록

* Restful API 지원으로 3rd party 제품과 연계 가능
