{/* 

#### 기능 개요

* **좌측 패널**: 등록된 디바이스 목록을 실시간으로 표시합니다. 각 디바이스는 NMS에 등록된 정보를 바탕으로 자동으로 업데이트됩니다.

* **우측 패널**: 선택한 디바이스 또는 인터페이스의 상세 지표 요약을 표시합니다. 이 화면에서는 주로 성능 지표나 상태 변화에 대한 그래픽을 제공합니다.

#### 주요 지표

* **디바이스별 지표**

  * ***Total Throughput***: 디바이스의 총 트래픽 양을 보여주는 지표입니다.

  * ***Inbound Throughput***: 수신된 트래픽의 양을 나타냅니다.

  * ***Outbound Throughput***: 송신된 트래픽의 양을 나타냅니다.

  * ***OID 기반 세션 정보***: 각 디바이스의 세션 수, CPU 사용률, 메모리 사용률 등을 표시합니다.
  
* **인터페이스별 지표**

  * ***Throughput***: 각 인터페이스의 트래픽 양을 나타냅니다.

  * ***Discard***: 네트워크 오류로 인해 폐기된 패킷 수를 나타냅니다.

  * ***Error***: 전송 중 오류가 발생한 패킷 수를 표시합니다.

  * ***PPS (Packets Per Second)***: 인터페이스에서 초당 전송된 패킷 수를 표시합니다.

#### 사용자 설정

* **지표 단위 설정**: 각 지표는 다양한 단위를 가질 수 있습니다. 예를 들어, 트래픽은 Mbps, 용량은 GB로 표시될 수 있습니다. NMS에서는 사용자에게 이러한 단위를 수동으로 설정할 수 있는 기능을 제공합니다.
